graphed-numpy
=============

A trivial numpy backend for ``graphed`` (milestone M2), proving the backend seam. Operates on 1-D
numpy bags: arithmetic, boolean filter, sum reduction, and opaque ``map`` callables (recorded as
External nodes with a preservation-risk payload descriptor).
